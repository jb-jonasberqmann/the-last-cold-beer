# THE LAST COLD BEER — Adobe Firefly Prompts
*Optimised for Adobe Firefly Image 3*

---

## FIREFLY SETTINGS (apply to every generation)

| Setting | Value |
|---|---|
| Content Type | Art |
| Style | Digital Art + Editorial Illustration |
| Visual Intensity | High |
| Negative prompt | (see per-prompt — always include) |

**Global negative prompt (add to every single generation):**
> cartoon, anime, 3D render, photorealistic photo, people, figures, persons, humans, text, watermark, logo, UI elements, borders, frames, busy background

**Style reference tip:** Once the first map or room background looks right, save it and upload it as a Style Reference in Firefly for every subsequent generation. This is the single most powerful tool for consistency.

## MAPS

*Aspect ratio: 2:3 (portrait). Content type: Art.*
*These are illustrated top-down views — like a hand-drawn treasure map or site plan. Nodes should be organic blob shapes. Paths should look hand-drawn.*

---

### MAP 1 — ACT 1 "THE ARRIVAL"

**Prompt:**
> top-down hand-illustrated map of a 1960s Danish summerhouse property, seen from above at a slight angle, golden hour afternoon light, warm amber and green colour palette, illustrated in the style of a vintage Nordic travel poster, hand-drawn organic blob shapes mark key locations: gravel driveway with a car, wooden terrace, large fenced garden with coastal grass, weathered tool shed, front door with a key box, petanque court, open carport, locations connected by hand-drawn illustrated paths like a treasure map, worn paper texture, no people, atmospheric, slightly eerie undertone beneath the warmth, editorial illustration style
